# Deck of Cards in Go
![built with GoLang](https://img.shields.io/badge/built_with-golang-blue.svg)

## Getting Started
`install go`

### Run app locally
`go run ./` or `go run main.go deck.go`

### Run tests
`go test`

#### Functions
`newDeck()`
`newDeckFromFile("nameOfFile")`
`deal(deck, card_in_hand)`
#### Receiver Functions
`d.print()`
`d.toString()`
`d.saveToFile()`
`d.shuffle()`
